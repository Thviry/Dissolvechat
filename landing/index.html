// HOW_WE_HANDLE_IT

Built to resist. Not to comply.

01

No accounts. No phone numbers.

Your identity is a cryptographic key pair stored on your device. No email, no phone, no server-side account. Back it up to a USB drive. Nobody can recover it — and nobody can seize it.

02

The server is a dumb pipe.

The relay routes encrypted blobs. It never sees plaintext, never stores keys, can't impersonate users. Ephemeral keys for forward secrecy on every message. The relay is designed to be untrusted — and replaceable.

03

You control who reaches you.

Capability-based addressing. To message you, someone needs a token you issued. Revoke any token instantly — cut off one sender without touching your inbox. No other messenger gives you this control.

04

Group chat. Same zero trust.

E2EE group conversations — AES-256-GCM symmetric key, wrapped per-member via ECDH. The relay never knows a group exists. Same security model as 1-to-1. Opt-in presence if you want it.

05

Open source. Fully auditable.

MIT licensed. Every line of code is public. Run your own relay if you don't trust ours. We built this because user security is in freefall and nobody seemed to care. We care.
